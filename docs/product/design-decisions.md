# Contäx デザイン意思決定

## 再構築方針（2026-04-07 確定）

### 何を変えるか
| 要素 | 変更内容 |
|------|---------|
| UIコンポーネント | shadcn/ui + Tailwind CSS で再構築 |
| カラー | 紫グラデ → ブルー単色（oklch(0.546 0.245 262)）に統一 |
| フォント | Noto Sans JP を適用 |

### 何を変えないか
| 要素 | 理由 |
|------|------|
| レイアウト・構造 | 現状のブロック構造はコアバリュー。触らない |
| アイコン | Fluent UI Emoji（Microsoft製3D風）を継続使用 |

### なぜこのデザイン方向か
- 現状の紫グラデUIが「AIツール感」を出しており、ターゲットの「ChatGPTと違う道具」という差別化を損ねていた
- Contäxは「文章のものさし・構成力を磨く道具」として位置づけるため、道具らしいクリーンなUIが必要
- shadcn/uiはプロダクト部署標準ライブラリ。保守性と一貫性のため採用

### 参考デザイン
- nani.now（https://nani.now/ja）: 3Dアイコン × クリーンな余白感のトーン&マナー
